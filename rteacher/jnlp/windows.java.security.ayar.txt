Java Güncellemesi sebebiyle "Your security settings have blocked a self-signed application from running" hatası alan kurumlar aşağıda belirtilen yöntemlerden birini uyguladıkları takdirde Kesenek uygulamalarına sorunsuz şekilde girebilirler.

Yöntem 1)
Denetim masasında bulunan JAVA alanında Security(Güvenlik) alanında girin
Security Level(Güvenlik Düzeyi) alanında Medium(Orta) seçeneğini şeçin
Apply(Uygula)'ya tıklayın.
Java Denetim Masası'nda yapılan değişiklikleri kaydetmek için Ok(Tamam)'a tıklayın.
(Konu ile ilgili link: Java Denetim Masası'ndaki güvenlik düzeyi ayarları)
